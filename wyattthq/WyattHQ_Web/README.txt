WYATT HQ FITNESS WEBSITE

Pages:
- index.html       Intro / homepage
- calculator.html  Separate calorie calculator page

Shared files:
- style.css        Styling and responsive layout for both pages
- script.js        Mobile navigation and calorie calculator logic
- images/hero-photo.png  The gym photo used on the home page

How to use:
1. Keep every file and the images folder together.
2. Open index.html in your browser.
3. The CALORIE SYSTEM navigation link opens calculator.html as a different page/tab.
4. To change your social links, update the Instagram and TikTok URLs in both HTML files.
5. To use a different photo, replace images/hero-photo.png with another image using the same filename.
